# Recommended Learning Path (8 Weeks)

## Weeks 1-2: ORM Foundations
- Start with Prisma official docs and the Deep-Dive Handbook
- Watch Prisma Full Course 2025 on YouTube
- Explore Drizzle ORM official tutorials and comparison articles
- Read TypeORM Getting Started and build a small project

## Weeks 3-4: Database Mastery
- Study PostgreSQL connection pooling with node-postgres
- Learn MongoDB with Mongoose TypeScript integration
- Read database design patterns and optimization articles
- Practice with the Database Design & Query Optimization GitHub repo

## Weeks 5-6: Design Patterns & SOLID
- Study Refactoring.Guru TypeScript patterns (start with Factory, Strategy, Observer, Decorator)
- Read SOLID principles articles (Strapi guide + LogRocket)
- Watch the 10 Design Patterns in 10 Minutes video, then deep-dive
- Implement patterns in a small TypeScript project

## Weeks 7-8: Architecture & DDD
- Study Clean Architecture articles and build a project using the boilerplate repos
- Learn Repository Pattern and Dependency Injection (tsyringe/InversifyJS)
- Explore DDD with Khalil Stemmler's articles and the ddd-forum repo
- Study microservices patterns (CQRS, Saga) with the booking-microservices repo
